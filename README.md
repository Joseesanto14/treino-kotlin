# Treino Kotlin 🚀

Repositório dedicado ao estudo e prática da linguagem **Kotlin**. Este projeto compila diversos exercícios práticos desenvolvidos a partir de materiais de apoio e apostilas, com foco em lógica de programação e **Programação Orientada a Objetos (POO)**.

## 📝 Descrição

O objetivo deste repositório é documentar a evolução no aprendizado de Kotlin, explorando desde conceitos básicos até tópicos avançados.

## 📚 Material de Apoio

Os exercícios seguem o cronograma da apostila:
- 📄 [Algoritmos e Lógica de Programação em Kotlin](./studybook/apostila_kotlin.pdf)
